# APLIKASI MANAJEMEN PORTOFOLIO DATA SCIENCE UNTUK PEMULA

Repositori ini berisi source code untuk Tugas Besar Algoritma dan Pemrograman (Alpro).
Aplikasi ini dirancang untuk membantu pemula dalam mengelola portofolio proyek Data Science
secara lebih terstruktur dan mudah dipahami.

## Fitur Utama
- 📂 Manajemen daftar proyek Data Science
- 📝 Input dan update detail proyek (judul, tanggal, kategori, teknologi, tingkat kesulitan)
- 🔍 Pencarian dan filter proyek berdasarkan kategori
- 📊 Ringkasan portofolio dalam format yang mudah dibaca
- 💾 Penyimpanan data sederhana menggunakan file

## Tujuan
Membantu mahasiswa atau pemula dalam Data Science untuk:
- Melatih keterampilan pemrograman dasar
- Membiasakan diri dengan struktur aplikasi sederhana
- Mengorganisasi portofolio proyek secara sistematis

## Teknologi
- Bahasa Pemrograman: GO
- Struktur data dasar (array)
